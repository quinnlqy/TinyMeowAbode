# 方寸喵居 - 玩家操作指南 (User Manual)

欢迎来到《方寸喵居》！这是一个温馨的治愈系放置游戏。在这里，你可以装修房间，照顾猫咪，享受慢节奏的云养猫生活。

## 1. 游戏界面与操作

### 💻 电脑端 (PC)
- **视角控制**：
  - **旋转**：按住 **鼠标右键** 拖动。
  - **平移**：按住 **鼠标中键** 拖动，或使用键盘 **W/A/S/D**。
  - **缩放**：滚动 **鼠标滚轮**。
- **家具互动**：
  - **放置**：从商店点击家具，移动鼠标选择位置，点击 **左键** 放置。
  - **旋转家具**：在放置或移动家具时，按键盘 **R** 键。
  - **移动已放置家具**：长按鼠标左键选中家具，拖动到新位置。
  - **删除家具**：选中家具后，按 **Delete** 键或点击 UI 上的删除按钮。

### 📱 手机端 (Mobile)
- **视角控制**：
  - **旋转/平移**：单指在空白处拖动（默认为旋转，部分模式下为平移）。
  - **缩放**：双指捏合/张开。
- **家具互动**：
  - **放置**：从商店拖拽家具图标到场景中。
  - **移动**：**长按** 场景中的家具 1 秒，进入移动模式，出现操作按钮（旋转/确认/取消）。
  - **旋转**：点击选中家具后出现的 **旋转按钮**。

## 2. 核心玩法

### ❤️ 赚取爱心
爱心是游戏里的货币，用于购买家具和猫粮。
1.  **抚摸猫咪**：点击猫咪身体，它会开心并掉落爱心（注意不要摸太久，它会生气哦！）。
2.  **每日登录**：每天第一次打开游戏，获得 **+50 爱心** 奖励。
3.  **自然产出**：猫咪使用家具（如睡觉、磨爪子）时会产生爱心。

### 🐱 照顾猫咪
猫咪有两项基本需求，可以在左上角状态栏查看：
- **饥饿度 (Hunger)**：
  - 当饥饿条变空时，猫咪会去找 **猫食盆**。
  - 如果猫食盆空了，点击它，花费 10 爱心补充猫粮。
- **如厕值 (Toilet)**：
  - 当状态条变红时，猫咪会去找 **猫砂盆**。
  - 使用后猫砂盆会变脏，点击它进行清理。

### 🏠 装修指南
- **商店**：点击右下角购物车图标打开。
- **分类**：分为家具、装饰（墙纸地板）、猫咪用品、特殊物品（盲盒）。
- **盲盒**：购买“马年盲盒”可随机开出隐藏款手办（烈焰小马、黑骏马等）。

## 3. 存档与日记

### 💾 存档系统
- **自动保存**：游戏每 30 秒自动保存一次。
- **手动保存**：点击设置里的 **"💾 保存"** 按钮，会下载一个 `.json` 文件到你的设备。
- **读取存档**：点击设置里的 **"📂 读取"** 按钮，选择之前的 `.json` 文件即可恢复进度。
  - *小贴士：如果你要在手机和电脑之间同步进度，可以使用这个功能传输文件！*

### 📖 猫咪日记
- 点击右上角的日记本图标。
- 这里记录了猫咪每天的趣事（抓蝴蝶、睡觉、看风景）。
- 离线期间的事件也会被补记在日记里。
