---
description: 提交游戏更新并触发部署 (Submit Game Update)
---

当用户说“提交吧”或类似指令时，执行以下步骤：

1. **检查版本号**: 
   - 查看 `index.html` 中的版本号 (例如 `V1.3`)。
   - 如果用户没有明确指定，默认将版本号 +0.1 (例如 `V1.4`)。
   - 修改 `index.html` 中的版本号显示。

2. **Git 提交**:
   - 运行 `git add .` 添加所有改动。
   - 运行 `git commit -m "update: v[新版本号] - [简要描述改动]"`。
     - 如果用户提供了描述，使用用户的描述。
     - 如果用户未提供，总结最近的修改作为描述。

3. **推送到远程**:
   - 运行 `git push`。
   - 这会自动触发 GitHub Pages 和 Cloudflare Pages 的构建与部署。

4. **通知用户**:
   - 告知用户提交已完成，部署通常需要几分钟。
